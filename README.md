# Calculadora Android

Um aplicativo de calculadora desenvolvido em Android com Kotlin e Jetpack Compose.

## Recursos

- ✅ Adição
- ✅ Subtração
- ✅ Multiplicação
- ✅ Divisão (com validação de divisão por zero)
- ✅ Porcentagem

## Como executar

1. Clone o repositório
2. Abra o projeto no Android Studio
3. Conecte um dispositivo Android ou inicie um emulador
4. Clique em "Run" para executar o aplicativo

## Estrutura do Projeto

```
app/src/main/java/com/example/calculadora/
├── CalculatorLogic.kt       # Lógica matemática
├── MainActivity.kt           # Atividade principal
├── components/
│   ├── ButtonY.kt           # Componente de botão
│   └── Teclado.kt           # Componente do teclado
├── screen/
│   └── CalculatorScreen.kt  # Tela principal da calculadora
└── ui/theme/
    ├── Color.kt             # Cores do tema
    ├── Theme.kt             # Configuração do tema
    └── Type.kt              # Tipografia
```
